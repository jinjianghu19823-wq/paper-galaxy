Future: personal cloud library

This is design-only. Paper Galaxy does not currently include cloud sync, hosted indexing, accounts, payments, telemetry, or document upload.

The proposed direction is staged: keep local-first as the default, then consider an opt-in encrypted backup vault before any metadata sync or managed compute.

Read design docs

Principles

User-owned library, no account required today, explicit upload boundaries, encryption-first storage, export/delete controls, and self-hosting considerations.

Cloud runtime is not implemented in this release.
