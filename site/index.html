Why Paper Galaxy?

It is a research cartography tool for people who want visual structure without turning a private library into a cloud product.

Local-first indexing

Scan local folders and store extracted records in SQLite under your project directory.

Dynamic graph

Browse clusters, focus neighbors, and inspect document metadata in a dependency-free browser UI.

Explainable neighbors

Nearest neighbors come from high-dimensional TF-IDF similarity, with visible terms and short chunk evidence.

Manual cluster labels

Rename generated labels locally while preserving the original evidence behind each cluster.

Optional embeddings

Dense vectors are explicit local commands, not a hidden download or remote default.

Export and backup

Validate projects, save map runs, and create local backups with clear sensitivity warnings.

Security and privacy

The local app has no account system, no telemetry, and no automatic upload. Optional OCR and embeddings run only when enabled.

Future cloud library

A personal cloud library is design-only today. Any future version must stay opt-in and keep the local workflow usable without an account.
