Privacy

Paper Galaxy is local-first by default. There is no account, telemetry, automatic upload, hosted backend, or remote labeling service.

What stays local

  • Source documents stay in your folders.
  • Extracted text, chunks, labels, vectors, and saved map runs live in local SQLite.
  • Graph positions and demo label edits use browser localStorage only.

Optional local features

OCR and dense embeddings are explicit local workflows. OCR may require a local Tesseract binary. Embedding commands require a local model path unless you explicitly allow model resolution.

Zotero boundary

Zotero import uses Zotero Desktop's local API. Paper Galaxy does not write to Zotero, performs no upload, and does not copy PDFs by default.

Backups are sensitive

Backup bundles may contain extracted text, chunks, vectors, labels, and map runs. Treat them like private research data.

Public demo

The GitHub Pages demo uses synthetic sample content only and does not talk to a backend.

Full privacy notes live in `docs/PRIVACY.md`.
