Install locally

Use an editable install while Paper Galaxy is still alpha. The app reads a local SQLite database and serves the browser UI on 127.0.0.1 by default.

Quickstart

python -m pip install -e ".[dev,ml,pdf,app]"
paper-galaxy doctor
paper-galaxy init . --force
paper-galaxy index examples/tiny_corpus --project-dir . --min-chars 40
paper-galaxy serve --project-dir .

Optional extras

`ocr` enables local OCR wrappers. `embeddings` enables explicit local dense-vector commands. Remote model names are rejected by default unless you opt in.

Local data warning

The SQLite database can contain extracted text, chunks, vectors, labels, and saved map runs. Keep `.paper-galaxy/` private.

Try the static demo before installing: open demo.
